1.0

  • New name (Programmer Enhancement Suite): No longer just "Daily Status Modifications".
  • Additionally, if you had the "PRx Dev Site Modifications" extension, it has been uninstalled, as this extension encompasses and supersedes its functionality.
  • Notifications have been completely rewritten to take advantage of Chrome's new rich desktop notifications. In the time warning dialog, you will be able to go directly to the Daily Status page or close the notification. Show Example
  • Other dev site mods in "Misc" tab in options (click Niblet's head):
    • "Go To Issue Number" Box on My Issues Page
      Displayed at the top of the Issue List page. Press the / key anywhere on the issue list page to activate. Additionally, paste anything into the box and non-numerics will be removed.
    • Omnibox Support! Type "nib" into your address bar and hit space. Hit space again to see options listed.
      You can go directly to many pages from the omnibox now: Daily Status, Issue Edit, My Issues, Server Search, Download Item Version Manage, Omnibox Help. It won't open a new tab if you already have it open; you will simply be transferred to that tab immediately!
    • Suppress popups (Issue Edit and/or All)
      Optionally suppresses Issue Edit and/or all popups on the site (Edit Issue, Edit Location, etc).
    • Copy without Silverlight
      Replaces Silverlight copy controls (copy server hostname, etc) with native copy functionality. It's much faster and more reliable, and does not require Silverlight to be installed.
    • Server search from Issue Edit
      Optionally suppresses Issue Edit and/or all popups on the site (Edit Issue, Edit Location, etc).
    • Display support person on Server Edit
      Optionally suppresses Issue Edit and/or all popups on the site (Edit Issue, Edit Location, etc).
    • Promote issue edit clickable spans to links
      Turns clickable "edit" <span>s into real links. You will be able to see the target, open in background tabs, copy the link, and everything you can do with normal links.
    • Link Issue Subject
      Turn Issue Subject into a clickable link. No more following the row to the left to click "Edit".
    • Display Server Search from Issue Edit
      Display Server Search from Issue Edit (useful for locations with multiple servers). "Multiple servers" becomes a link which takes you to the Server Edit page and automatically searches the appropriate LocationID.
