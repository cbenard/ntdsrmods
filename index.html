Daily Status Modifications Extension (ntdsrmods)

By Chris Benard

Information

This chrome extension is for use internally on a time clock web page. If you don't know what it's for, it's probably not for you.

Download and Installation

Follow these steps precisely

  1. You need to open your chrome extensions settings page. You can get to this by typing chrome://extensions into the address bar, or currently (Chrome's UI changes a lot) by going to the menu icon at the top right , selecting Settings, and then selecting extensions. Good luck!
  2. Then download the ntdsrmods-xxx.crx (where xxx is a version) file to your computer.

  3. Finally, drag the extension file into the extensions page and you should see the image below. If you don't, it's not going to work and you're going to get a warning about not being able to install from websites.

    This also will not work if you spawn the settings page into its own window to make it easier. Google hates you; I'm sorry.

Usage

What You Should See

Options

Click on Niblet's head to adjust settings. The changes should be reflected immediately in Daily Status tabs. The "Beginning of Day" setting is unused right now, but set it correctly, because it will be used in the future. Enjoy!
