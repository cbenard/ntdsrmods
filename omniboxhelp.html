Omnibox Help

Available Commands

  • dsr » Goes directly to Daily Status
  • issues or issue » Goes directly to Issue List
  • issue 123456 » Goes directly to Issue #123456
  • servers » Goes directly to Server Search
  • help » Goes directly to this help page

In all cases, you can shorten the commands as long as they are non ambiguous:

For example:

  • d » Daily Status
  • i » Issue List
  • i 123456 » Issue #123456
  • s » Server Search
  • hel » Satan's Lair
