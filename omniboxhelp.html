Available Commands

  • calendar » Goes directly to the calendar
  • divpermissions » Goes directly to Manage Download Item Version Permission
  • divsearch » Goes directly to Download Item Version Search
  • dsr » Goes directly to Daily Status
  • issues » Goes directly to Issue List
  • issue 123456 » Goes directly to Issue #123456
  • locations » Goes directly to Location Search
  • location [Location Name] »
    Goes directly to Location Search and searches for the given name (do not use square brackets)
  • location [LocationID GUID] »
    Goes directly to Location Edit of LocationID specified (do not use square brackets)
  • servers » Goes directly to Server Search
  • servers Somewhere » Goes directly to Server Search and searches for "Somewhere"
  • help » Goes directly to this help page

In all cases, you can shorten the commands as long as they are non ambiguous:

For example:

  • c » Calendar
  • divp » Manage Download Item Version Permission
  • divs » Download Item Version Search
  • ds » Daily Status
  • i » Issue List
  • i 123456 » Issue #123456
  • L » Location Search
  • L test » Location Search for "test"
  • L 2773c554-fc77-4067-94bd-71bae5bcd2b5 »
    Location Edit for LocationID 2773c554-fc77-4067-94bd-71bae5bcd2b5
  • s » Server Search
  • s Somewhere » Server Search for "Somewhere"
  • hel » Satan's Lair
